Colabore

Que bom que você se interessou em colaborar com o projeto! =)
Veja aqui algumas maneiras possíveis:

Tecnicamente

Desenvolver e manter a plataforma e novos scripts de libertação de dados é de suma importância para que o projeto seja bem sucedido. Entre em nosso chat para conversarmos melhor sobre como você pode nos ajudar!.

Aqui estão listadas algumas ações bem simples, mas que são de grande ajuda:

  1. Utilizar e citar os dados disponíveis no site
  2. Sugerir um novo dataset para libertarmos
  3. Nos contar os erros que você encontrar ao utilizar a plataforma
  4. Sugerir coisas interessantes que você acha que irão contribuir para o projeto!
Financeiramente

Possuímos diversos gastos para manter o site online e desenvolver novas funcionalidades e você pode também contribuir para a sustentabilidade financeira do projeto. Se tiver interesse, entre em contato.

  1. Encomendando a libertação de um novo dataset;
  2. Encomendando análises e visualizações complexas nos dados que já temos disponíveis;
  3. Doando recorrentemente.
